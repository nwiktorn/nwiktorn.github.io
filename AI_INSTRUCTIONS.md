# AI Collaboration Guidelines

This document contains critical rules for AI assistants working on this repository. Please read and follow them strictly.

## 1. Deployment Rule (CRITICAL)
**NEVER** execute `git push` or deploy the project online automatically. All changes must remain local until a deployment is explicitly requested.

## 2. Preview Rule
After introducing any changes, always provide a link to the local preview:
👉 [**http://localhost:8080**](http://localhost:8080)

## 3. Approval for Online Deployment
Deploy to GitHub Pages/Production ONLY when the user provides one of the following explicit commands:
- "Wrzuć online"
- "Wrzuć na GitHub"
- "Wrzuć na serwer"
- "Zrób deployment"

## 4. Technology Stack
- Vanilla HTML/CSS/JS (No frameworks unless requested).
- Design focus: High-end, premium aesthetics, smooth animations.
